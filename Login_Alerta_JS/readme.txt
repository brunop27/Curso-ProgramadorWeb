#login.html

O documento apresenta uma interação de uma tela de login, por meio de JavaScript. 
É criado um elemento na página, por meio de javascript, indicando se houve 
sucesso ou erro no login. No JavaScript foram instanciadas algumas variáveis para
capturar e manipular os elementos do HTML e também foram criadas funções que criam e retorna a mensagem
"sucesso" ou "Login invalido". Conhecimentos utilizados: 

JavaScript
- .getElementsByTagName()
- .createElement()
- .classList.add()
- .appendChild()
- .insertAdjacentElement()
